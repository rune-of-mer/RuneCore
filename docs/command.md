# Package org.lyralis.runeCore.command

RuneCore で使用できるコマンドの一覧

- `[]`: オプション値．指定せずとも実行可能です．
- `<>`: 必須値．実行するには指定が必要です．

### `/dice [最大値] [出力モード]` - `v0.1.0~`

0～999の数字の1つをランダムに抽選して，全体チャットもしくは自分に対して結果を発言します．

省略した場合は，0〜999の数字で抽選・全体チャットで発言します．

### `/logout` - `v0.1.0~`

サーバーからログアウトします．飛行中の場合はログアウトできません．

### `/patchnote` - `v0.1.0~`

パッチノートへの URL を表示します．

### `/playerlist` - `v0.1.0~`

現在接続しているプレイヤーのリストを表示します．

### `/playtime` - `v0.1.0~`

現在までの累積プレイ時間を表示します．
