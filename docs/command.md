# Package org.lyralis.runeCore.command

RuneCore で使用できるコマンドの一覧

- `[]`: オプション値．指定せずとも実行可能です．
- `<>`: 必須値．実行するには指定が必要です．

### `/dice [最大値] [出力モード]`

0～999の数字の1つをランダムに抽選して，全体チャットもしくは自分に対して結果を発言します．

省略した場合は，0〜999の数字で抽選・全体チャットで発言します．

### `/logout`

サーバーからログアウトします．飛行中の場合はログアウトできません．

### `/patchnote`

パッチノートへの URL を表示します．

### `/playerlist`

現在接続しているプレイヤーのリストを表示します．

### `/playtime`

現在までの累積プレイ時間を表示します．

### `/experience add [経験値] [プレイヤーID]`

指定したプレイヤーに対して経験値を付与します．

省略した場合は `500` の経験値を自分に付与します．

### `/level`

現在のレベル情報を表示します
